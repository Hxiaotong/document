# NVM【node的版本管理工具】
## 常用命令
1. 安装最新稳定版
`node nvm install stable ` 
2. 安装指定版本
`nvm install <version>，如：安装v4.4.0，nvm install v4.4.0`
3. 删除已安装的指定版本，语法与install类似
`nvm uninstall <version>  `
4. 切换使用指定的版本node
`nvm use <version> ` 
5. 列出所有安装的版本
`nvm ls`
6. 设置默认版本
`nvm alias default <version>  如： nvm alias default v11.1.0`
